# Turtle Crossing Game

A small 2D Python game where the player controls a turtle and must cross a road while avoiding moving cars.

## Features

- Object-oriented architecture (Player, CarManager, Scoreboard)
- Collision detection using distance calculations
- Dynamic difficulty system (increasing car speed and spawn rate)
- Dark mode with radius-based visibility system
- Keyboard-controlled player movement

## Technologies

- Python
- Turtle graphics library

## Learning Objectives

This project helped me practice:
- object-oriented programming
- game loop design
- event handling
- modular program structure
